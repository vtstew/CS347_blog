Hidding DOM Elements (Blog #2)

This week I chose to do research on hiding DOM elements. There are different ways to hide elements depending on what you want the element hidden from. These options are: completely hidden, semantically hidden, and visually hidden. (see table below)

visibility state on screen in accessibility tree keyboard navigation
completely hidden hidden hidden not navigatable
semantically hidden visible hidden navigatable
visually hidden hidden visible navigatable

Completely Hidden:

There are three different options for completely hiding elements. Two options are CSS properties and one is an html property. Although all these options do the same thing, they will do so in different ways.

YOU FOUND IT!

CSS Property Options:

  • Display: none
    • Completely vanishes
    • Takes no space on page
    • Won't allow animation
  • Visibility: hidden
    • Completely vanishes
    • Preserves space on page
    • Allows animation
    • Has ability to unhide nested content

HTML5 hidden attribute:

  • Element.hidden = true;
    • this command causes the browsers internal style sheet to hide the element
