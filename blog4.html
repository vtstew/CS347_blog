Why Do We Use Web Animations?

This week I read about the purpose of web animations. Mainly, they are used to attract users and improve a website's usability. Accoring to one artical, there are five main reasons that a buisness would want to include anaimations in their websites design:

Attracting Attention

People want to be entertained. Animations make using a website more interesting, thus drawing the users attentions and attracting more users.

Guiding Users Through a Process

Animations can tell the user where they should be going/looking next as they navigate the website.

Engaging the User

Animations can tell a story that keeps users engaged for longer.

Communicating An Action

Animations help with instructions that a user may need to follow. It can make it easier for a user to understand and follow the directions.

Cleaning Up A Layout

Animations can either display or remove elements that are not always necessary to be on the page.
